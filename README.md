# AI Learning Notes

My personal notes following **Incubyte's AI Learning Planner**.

## Structure

```
part1-basics/
  week1/
    day1/ - LLM Fundamentals & Prompt Engineering
    day2/ - AI Ethics, Security & Responsible Use
    ...
```

Each day includes:
- **Lessons** (3 min reads) - Core concepts
- **Exercises** - Hands-on practice
- **Resources** - Links and references
- **Reflection** - Personal takeaways

## Quick Start

1. Navigate to the current week/day
2. Read lessons in order (01, 02, 03...)
3. Complete exercises
4. Fill out reflection
